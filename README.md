# Fuzzy Washer Controller
**Fuzzy Logic Control System for a Washing Machine in Java**

This project implements an intelligent control system based on **Fuzzy Logic** to simulate the automatic behavior of a washing machine.  
The system dynamically adjusts parameters such as washing time, water level, and washing intensity based on input conditions.

---

## Features

- Fuzzy controller implemented in Java.
- Membership function definitions.
- Fuzzy rule base.
- Inference engine and defuzzification.
- Automatic decision simulation.
- Modular and extensible architecture.
---

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Fuzzy Logic
- Recommended IDE: IntelliJ IDEA / NetBeans / Eclipse
